# FightingMates

Juego de estrategia por turnos para terminal en el que dos jugadores se enfrentan usando ataques, habilidades y decisiones tácticas para derrotar al rival.

## 1) Descripción del juego
- **Nombre del juego:** FightingMates
- **Género:** Estrategia por turnos
- **Objetivo principal:** Reducir los puntos de vida del oponente hasta ganar el combate.
- **Mecánicas principales:**
  - Atacar.
  - Usar habilidades.
  - Gestionar cambios de cartas/acciones según la situación de la partida.
- **Público objetivo:** Público general.
- **Plataforma objetivo:** Terminal.

## 2) Repositorio y ramas de trabajo
- **Repositorio:** https://github.com/pepeluaybar/FightingMates
- **Rama `main`:** Rama estable y de entrega final.
- **Rama `dev`:** Rama de integración para revisión docente.
- **Ramas de tareas:**
  - `tarea-1/[nombre-o-id]`
  - `tarea-2/[nombre-o-id]`

### Flujo de trabajo con Pull Request
1. Crear una rama de tarea desde `dev`.
2. Implementar cambios y realizar commits claros.
3. Abrir un Pull Request desde la rama de tarea hacia `dev`.
4. Revisar, corregir y aprobar el PR.
5. Hacer merge a `dev`.
6. Cuando `dev` esté validada por la profesora, hacer merge de `dev` a `main`.

> **Nota:** El código que se presenta para aprobación es el de la rama `dev`.
